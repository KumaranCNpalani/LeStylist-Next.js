## For Website Admin

### How to Update Content

#### 1. Add/Edit Services
1. Login to admin: `yourdomain.com/admin/login`
2. Go to **Services**
3. Click **Add Service** or **Edit** existing
4. Fill in details and save
5. ‚úÖ Service appears on website instantly!

#### 2. Upload Gallery Images
1. Go to **Admin ‚Üí Gallery**
2. Click **Upload Image**
3. Select category:
   - `awards` = Awards section on About page
   - `shop` = Experience section on About page
   - `hair` = Hair gallery
   - `makeup` = Makeup gallery
   - `skin` = Skin care gallery
   - `client` = Client photos gallery
4. Upload and save
5. ‚úÖ Image appears on website instantly!

#### 3. Manage Testimonials
1. Go to **Admin ‚Üí Testimonials**
2. Click **Add Testimonial**
3. Fill in customer details and review
4. Mark as **Approved** to show on website
5. ‚úÖ Testimonial appears on home page!

#### 4. Create Offers (Shows Banner on Home Page)
1. Go to **Admin ‚Üí Offers**
2. Click **Add Offer**
3. Fill in:
   - Title (e.g., "Summer Special")
   - Description
   - Discount type (Percentage or Fixed Amount)
   - Discount value
   - Valid dates
4. Set **Active** to "Yes"
5. ‚úÖ Offer banner appears on home page!
